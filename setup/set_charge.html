충전기 설정

  • Offline 상태일때 로컬인증 리스트 혹인 인증 캐쉬에 등록되어 있지 않은 사용자 충전 허용 여부설정
  • 인증캐쉬(Authorization Cache) 허용 여부설정
  • 리모트 충전이 시작하기 전 사용자에 대한 인증 여부 설정
  • 신호를 전달할 때 충전기 조명이 깜작이는 횟수 설정
    회
  • 시계 정렬된 데이터를 수신하는 간격을 설정, 0은 시계 정렬된 값을 전송하지 않음을 설정
    초
  • 충전 케이블 커넥터가 올바르게 삽입하지 않아 충전이 자동으로 취소 되기까지의 간격을 설정
    초
  • 충전기가 Heartbeat PDU를 보내야 하는 간격을 설정
    초
  • 충전기의 조명을 켤때 그 최대 강도 설정(백분율로 표시)
    %
  • 충전기가 오프라인 상태일 때 로컬 인증된 식별자를 사용하여 충전을 시작할지 여부 설정
  • 충전기가 온라인 일 때 중앙 시스템으로부터 인증을 기다리거나 요청하지 않고 로컬 인증된 식별자를 사용하여 충전을 시작할지 여부 설정
  • 충전 시작 후 중앙 시스템에서 사용자 인증이 무효화된 경우 전달되는 최대 에너지랭(wh)을설정
    wh
  • 충전기의 각종 측정값(전류량, 전력량, 전략세기, 충전진행율 등)을 전달하는 간격을 설정
    초
  • 충전기의 상태정보를 중앙 시스템으로 전송되기 전에 충전기나 커넥터 상태가 안정적으로 유지되어야 하는 최소기간 설정
    초
